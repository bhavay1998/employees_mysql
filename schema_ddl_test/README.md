# Schema - DDL Test
This folder contains files for a newly generated 'sales' database in MySQL. This database is only a sample database, and is <i>NOT</i> the primary database used to conduct analysis. The purpose of creating a sample database is to test the key elements of the <i> Data Definition Language </i> (DDL). 

## Folder Files
- The relationship schema can be found in the "sample_db_schema.jpg" image file.
- The "Creating_a_database.sql" file contains the actual code used to prepare the database and test-out <i>DDL</i>.

## About the <i>sales</i> database
The <i>sales</i> database contains 4 tables:
-<i>customers</i>
-<i>companies</i>
-<i>products</i> 
-<i>sales</i>

The <i>sales</i> table contains sales records of a retailer who specialises in consumer products (CPG retailer).
